
# 🚀 30 Days of DSA Mastery: Bootcamp & Challenge

A dedicated repository for my 30-day intensive sprint to master **Data Structures and Algorithms**. This bootcamp focuses on building the problem-solving intuition required for SDE roles at top-tier product companies.

## 🎯 Objectives

  * **Consistency:** Solve problems daily for 30 days without breaks.
  * **Efficiency:** Focus on optimizing Time Complexity ($O$) and Space Complexity.
  * **Documentation:** Not just code, but explaining the *why* behind every approach.
  * **Interview Readiness:** Specifically targeting patterns seen in August/September campus placements.

-----

## 📅 Roadmap & Curriculum

| Week | Focus Area | Key Topics |
| :--- | :--- | :--- |
| **Week 1** | **Arrays & Hashing** | Kadane’s, Two Pointers, Sliding Window, HashMaps |
| **Week 2** | **Linked Lists & Stacks** | Fast/Slow Pointers, Recursion, Monotonic Stacks |

## 🛠️ Daily Workflow

1.  **Understand:** Read the problem statement and identify the constraints.
2.  **Brute Force:** Write down the most basic solution first.
3.  **Optimize:** Apply DSA patterns to reduce complexity (e.g., using a HashMap to turn $O(n^2)$ into $O(n)$).
4.  **Reflect:** Update the `Logic.md` for that day with what I learned.
